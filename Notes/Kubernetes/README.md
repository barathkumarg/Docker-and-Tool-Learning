# Kubernetes 

## Core Concepts

- **ClusterArchitecture**: A Kubernetes cluster consists of a control plane and worker nodes. The control plane manages the cluster, while worker nodes run the applications.
- **API Primitives**: Kubernetes uses API primitives such as Pods, Services, Deployments, and ConfigMaps to manage applications and resources.
- **Service and Other Network Primitives**: Kubernetes provides various network primitives to manage communication between applications, including Services, Ingress, and Network Policies.
